# FrontPorted
FrontPorted is a fabric mod for 1.16.5 that "front-ports" features from older mods (e.g. ToggleSprint or Keystrokes) and also adds some custom QOL features.

#### Features:
- ToggleSprint
- ToggleSneak
- FullBright
- Block Overlay
- Better Animations
- Chat Timestamps
- Kill Sounds
- Better Chat Background
- KeyStrokes (planned)
- ...
